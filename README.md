# Проект BooksCollector

## Приложение позволяет
- Добавлять книги;
- Задавать жанр книг;
- Получать списки книг;
- Формировать списки книг, подходящих детям;
- Работать со списком избранных книг.
## Реализованные Тесты
- `add_new_book`  
  - проверка добовления книги с корректным названием;
  - проверка невозможности добавить книгу с пустым названием и именем длиннее 40 символов;
  - проверка, что одна и та же книга не добавляется в словарь дважды.

- `set_book_genre` и `get_book_genre`  
  - установка жанра только из допустимого списка жанров;
  - проверка, что у только что добавленной книги жанр по умолчанию отсутствует (пустая строка).

- `get_books_with_specific_genre`  
  - проверка, что возвращаются только книги с указанным жанром.

- `get_books_genre`  
  - проверка, что метод возвращает актуальный словарь книг и их жанров.

- `get_books_for_children`  
  - проверка, что книги с жанрами с возрастным рейтингом не попадают в список книг для детей.

- `add_book_in_favorites`, `delete_book_from_favorites`, `get_list_of_favorites_books`  
  - проверка добавления книги в избранное;
  - проверка, что нельзя добавить одну и ту же книгу в избранное дважды;
  - проверка удаления книги из избранного и получения актуального списка избранных книг.
